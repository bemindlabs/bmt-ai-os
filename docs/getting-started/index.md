# Getting Started

Welcome to BMT AI OS. This section walks you through getting the system running, from installing on real hardware to spinning up a development stack on your laptop.

## Paths

| Path | Time | When to use |
|------|------|-------------|
| [Quick Start](quick-start.md) | ~5 min | Already running BMT AI OS on hardware |
| [Installation](installation.md) | ~30 min | Flashing to a Tier 1 board |
| [Dev Setup](dev-setup.md) | ~10 min | x86 / macOS development with Docker |

## Prerequisites

- **Hardware:** ARM64 board with 8GB+ RAM, or any machine with Docker for dev setup
- **Network:** Internet access for initial model downloads; fully offline after that
- **Storage:** 32GB minimum; 64GB+ recommended for Standard/Full model presets

## Recommended First Steps

1. If you have a supported board, follow the [Installation Guide](installation.md) to flash the OS image.
2. If you want to develop or test without hardware, start with [Dev Setup](dev-setup.md).
3. Once the stack is running, follow the [Quick Start](quick-start.md) to pull a model and run your first query.
